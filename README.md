# 自己紹介サイト

開発者の名刺的なシンプルな自己紹介サイト。

## 概要

Vercel で公開する自己紹介サイトを作成します。
名刺的に、URL を共有して自己紹介できるようにします。

## 要件

### 必須機能

- [x] 名前 / ハンドルネームの表示
- [x] 簡単な自己紹介文
- [x] SNS リンク（GitHub, Twitter）

### あると嬉しい機能

- [x] スキル・使える技術の表示
- [x] プロフィール画像
- [x] ダークモード対応
- [x] アニメーション効果

## 使用技術

- React
- Tailwind CSS
- Vercel

## デザイン

### コンセプト

以下を取り入れたデザイン
- 動きがある（アニメーション）
- グラスモーフィズム
- グラデーション

### 配色

下記2色をメインで必要に応じて明度や彩度を変えて使用（背景なら淡く、文字なら濃く）
- プライマリー: #f9d2e8
- セカンダリー：#8772c0
